【内容紹介】
  javaでgifアニメーション(gif animation)を作成するAPIです。

【ファイルの内容】
　readme.txt :当テキストファイル
　lib\GifAnimationEncoder.jar :バイナリファイル。クラスパスをこのファイルに
                               張ってください。
　source     :ソースファイル
  sample     :簡単なgifアニメーション生成サンプル
  javadoc    :javadoc

【使い方】
  sample, javadocを見てください。

【注意事項】
  256色より多いイメージは、生成できません。例外IOExceptionを投げます。
